Releases history

This is the plain history of all releases of Tagging Plugin!

1.3 Series

  • 1.3.3 It is PHP5 compliant (passing by reference is deprecated) and bundles some bug-fixes (case sensitivity compare issue and [code] tag not being stripped)
  • 1.3.2 It fixes a bug with tags in post that are in 2nd page or more, it continued to display the first page of thread
  • 1.3.1 General bug fix release and fixes a serious bug with regular expression
  • 1.3 Ported to MyBB 1.6, it brings some code improvements and it fixes some bugs with hooks (tid argument not properly passed) and added a new hook to intercept merging of tags

1.2 Series

  • 1.2.2 Search results are parsed. {posturl} variable jumps directly to the post. Small database change, tags column from VARCHAR to TEXT (use Tagging Backup! for backupping tags before updating and then use the .sql file it generates)
  • 1.2.1 Fixed an error while updating post
  • 1.2 Finally a stable version is here, the system has been rethought for the last time, now it is very efficient, i added some plugin hooks which allow developers to extend the plugin without code edits

1.1 Series (beta)

  • 1.1 A completely rethought system and new features like a section in UserCP for tag settings (if i don't want to be tagged and similar)

1.0 Series (beta)

  • 1.0.1 Just a bugfix release that addressed issues reported by frostschutz.
  • 1.0 The first (and very very buggy) version, it relied completely on string analysis, very obsolete algorithm.
