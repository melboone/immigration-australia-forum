Developers

In this section i will cover everything you need to know for extending and translating this plugin.

Hooks

There are 3 hooks:
  • tagging_tag_added This hook is called when new tags in a post are added and it has an argument which is an array('pid' => $pid, 'tid' => $tid, 'users' => $users):
    pid: is the post id where users are tagged
    tid: is the thread id which contains the post
    users: is an array of user ids of the tagged users
  • tagging_tag_removed This hook is called when some tags are removed in a post and it has an argument which is an array('pid' => $pid, 'tid' => $tid, 'users' => $users):
    pid: is the post id where users are tagged
    tid: is the thread id which contains the post
    users: is an array of user ids of the removed tagged users
  • tagging_tag_merged This hook is called when tags are merged (when posts containing tags are merged) in a post and it has an argument which is an array('pid' => $pid, 'tid' => $tid, 'users' => $users):
    pid: is the post id which contains all merged pots
    tid: is the thread tid which contains the post
    users: is an array of user ids contained in tags

Translation

English language pack is self-contained, for translating it you need to create a file in /inc/languages/yourlanguage/ and name it tagging.lang.php.
This is the basic template of the file:
<?PHP
$l['tagging_usercp_noone'] = "No one";
$l['tagging_usercp_all'] = "Everyone";
$l['tagging_usercp_buddy'] = "Only your buddies";
$l['tagging_usercp_settings'] = "Tagging settings";
$l['tagging_usercp_update_settings'] = "Update your tagging settings";
$l['tagging_usercp_setting_invalid'] = "We're sorry but the setting is invalid, retry!";
$l['tagging_usercp_setting_updated'] = "You updated successfully your tagging settings";
?>
