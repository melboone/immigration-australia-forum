Installation

Installing is simple, just upload tagging.php into /inc/plugins/ directory of your MyBB copy and then activate it from Admin CP.

Usage

It is very simple in post write this:
@[object]
Object can represent or the uid of the user or the username and then in post you will see the tag and the tagged user will be warned of the tag via PM.

Tagging settings

What if an user doesn't want to be tagged from everyone or not at all or only from his buddies?!
Well in UserCP there is a section "Tags" where you can set who is allowed to tag you!

Upgrade

If a new version bundles some database change use Tagging Backup! plugin and backup all tags.
Then uninstall the plugin.
Upload the new tagging.php file and then activate and use the .sql file generated by Tagging Backup! plugin.
